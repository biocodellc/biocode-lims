Moorea Biocode LIMS Plugin

The Biocode software comprises everything you need to manage your lab and sequence analysis workflows. See the official wiki at https://github.com/biocodellc/biocode-lims/wiki for more information on how to use the plugin.

Release notes are available at https://github.com/biocodellc/biocode-lims/releases
